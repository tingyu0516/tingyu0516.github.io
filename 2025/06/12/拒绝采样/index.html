什么是拒绝采样

当我们面对一个"黑盒"函数时，即我们无法直接获取其数学表达式，但可以通过输入值得到对应的输出值，我们该如何去理解和近似这个函数呢？拒绝采样（Rejection Sampling）就是一种解决这类问题的有效方法。它通过构造一个已知的、易于采样的概率分布来近似目标分布，从而帮助我们理解和模拟这个"黑盒"函数的行为。这种方法的核心思想是：通过一个简单的、已知的分布来"包裹"目标分布，然后通过接受或拒绝采样的方式来获得符合目标分布的样本。

采样原理

假设我们有一个目标概率密度函数q(x)q(x)q(x)，为了对其进行采样，我们可以选择一个已知的、易于采样的建议分布（proposal distribution）p(x)p(x)p(x)，比如均匀分布或高斯分布。然后，我们需要找到一个合适的缩放常数MMM，使得Mp(x)Mp(x)Mp(x)能够完全覆盖目标函数q(x)q(x)q(x)，即满足q(x)≤Mp(x)q(x) \leq Mp(x)q(x)≤Mp(x)对所有xxx都成立。

这个缩放常数MMM的选择是一个关键步骤：

  1. MMM必须足够大，以确保Mp(x)Mp(x)Mp(x)能够完全覆盖q(x)q(x)q(x)，这是保证采样正确性的必要条件
  2. 但MMM又不能过大，因为过大的MMM会导致采样效率降低，增加计算开销

在实际应用中，我们通常希望MMM尽可能小，这样可以提高采样效率，同时又能保证采样的正确性。这个平衡点的选择往往需要根据具体问题来调整。
采样的具体步骤如下：

  1. 从建议分布p(x)p(x)p(x)中随机采样一个点xxx
  2. 计算接受概率α=q(x)Mp(x)\alpha = \frac{q(x)}{Mp(x)}α=Mp(x)q(x)​
  3. 生成一个均匀分布的随机数u∼U(0,1)u \sim U(0,1)u∼U(0,1)
  4. 如果u≤αu \leq \alphau≤α，则接受这个样本xxx；否则拒绝这个样本，重新从步骤1开始

这个过程可以形象地理解为：我们在Mp(x)Mp(x)Mp(x)曲线下方随机投点，如果点落在q(x)q(x)q(x)曲线下方，则接受这个样本；否则拒绝这个样本。通过这种方式，我们最终得到的样本将服从目标分布q(x)q(x)q(x)。

需要注意的是，采样效率（即接受率）与MMM的选择密切相关。MMM越大，接受率越低，采样效率也就越低。因此，在实际应用中，我们需要在保证采样正确性的前提下，尽可能选择较小的MMM值。

具体例子

让我们通过一个简单的例子来理解拒绝采样的过程。假设我们有一个离散的目标概率分布q(x)q(x)q(x)，其中xxx的取值范围是1到10的整数。这个分布有10个概率值，它们的和为1。我们可以用一个均匀分布p(x)=1p(x)=1p(x)=1作为建议分布，此时M=1M=1M=1就足够了，因为q(x)≤1q(x) \leq 1q(x)≤1对所有xxx都成立。

假设这10个概率值分别是[0.1, 0.15, 0.2, 0.1, 0.05, 0.1, 0.1, 0.05, 0.1, 0.05]。我们计划进行10,000次采样，具体步骤如下：

  1. 从均匀分布U(1,10)U(1,10)U(1,10)中随机生成一个整数xxx
  2. 计算接受概率α=q(x)\alpha = q(x)α=q(x)
  3. 生成一个均匀分布的随机数u∼U(0,1)u \sim U(0,1)u∼U(0,1)
  4. 如果u≤αu \leq \alphau≤α，则接受这个样本xxx；否则拒绝这个样本，重新从步骤1开始

通过这种方式，我们最终得到的样本将近似服从目标分布q(x)q(x)q(x)，也就是最终的样本数将近似为[100, 150, 200, 100, 50, 100, 100, 50, 100, 50]。这个例子虽然简单，但很好地展示了拒绝采样的核心思想：通过一个简单的均匀分布来"包裹"目标分布，然后通过接受或拒绝的方式来获得符合目标分布的样本。在实际应用中，这种方法的效率取决于建议分布与目标分布的相似程度，以及缩放常数MMM的选择。当建议分布与目标分布越接近，MMM值越小，采样效率就越高，此时的M值最好设置其实是0.1但是我们预先不知道。

大模型的拒绝采样

在大模型领域，拒绝采样同样具有重要的应用价值。让我们设想一个理想场景：假设存在一个理论上最优的大语言模型，我们将其表示为q(x)q(x)q(x)。这个最优模型虽然理论上存在，但由于其参数空间过于复杂，我们无法直接获取或表示其具体形式。在这种情况下，拒绝采样提供了一种可能的解决方案。

具体来说，我们可以使用一个已知的、可访问的大模型p(x)p(x)p(x)作为建议分布。对于给定的提示词，我们让这个已知模型生成多个候选回答。然后，通过一个评价模型对这些回答进行质量评估，将质量不达标的回答拒绝，只保留高质量的答案。理论上，通过这种方式，我们可以获得接近最优模型q(x)q(x)q(x)的生成结果。

然而，这种理想情况在实际应用中面临着诸多挑战：

  1. 覆盖性问题：根据拒绝采样的基本原理，我们需要确保已知模型p(x)p(x)p(x)的生成空间能够完全覆盖最优模型q(x)q(x)q(x)的生成空间。这要求已知模型具有足够强的生成能力。

  2. 评价模型的准确性：评价模型需要能够准确判断生成答案的质量，这本身就是一个极具挑战性的任务。评价模型的偏差或错误可能会影响整个采样过程的有效性。

  3. 采样效率：如果已知模型与最优模型的差距较大，或者评价标准过于严格，可能会导致采样效率低下，需要生成大量候选答案才能获得少量有效样本。

  4. 计算资源消耗：生成多个候选答案并进行评价需要消耗大量的计算资源，这在实际应用中可能成为限制因素。

因此，虽然拒绝采样为大模型优化提供了一种理论上的可能路径，但要将其转化为实际可行的解决方案，还需要在模型架构、评价方法、采样策略等多个方面进行深入研究和改进。这个领域仍然有很长的路要走，需要研究者们的持续探索和创新。
